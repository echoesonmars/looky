### 🟢 Низкая сложность (Базовый функционал, логика и парсинг)

* **Трекер износа (Cost Per Wear):** CRUD-архитектура с простой математикой. Пользователь отмечает использование вещи, а система считает «стоимость одной носки» (цена / количество надеваний), помогая анализировать окупаемость гардероба.
* **Умный трекер цен (Price Drop Alerts):** Настройка парсеров (cron-задачи) или использование API магазинов. Отслеживание конкретных товаров и отправка Push-уведомлений при снижении цены до заданного порога.
* **Tinder для одежды (Геймификация):** Реализация карточного UI со свайпами и базовой системы коллаборативной фильтрации. Алгоритм собирает датасет предпочтений (лайк/дизлайк) для улучшения рекомендаций.

### 🟡 Средняя сложность (Интеграция LLM, базовое компьютерное зрение и NLP)

* **Auto-Tagging при загрузке:** Использование легковесных моделей сегментации (удаление фона) и классификации изображений (ResNet/CLIP). Система автоматически распознает категорию вещи, цвет, сезон и стиль по фотографии с телефона.
* **Эко-скоринг ("Умный состав"):** Использование NLP (обработки естественного языка) для парсинга и анализа текстового описания состава. Система оценивает долговечность ткани и предупреждает о минусах (например, образовании катышков).
* **Сбор капсулы по бюджету:** Алгоритм комбинаторной оптимизации (поиск оптимального набора) плюс интеграция с парсером цен. Система подбирает готовый образ из разных магазинов строго под лимит пользователя с учетом скидок.
* **AI-Стилист (Умный чат):** Связка LLM (например, OpenAI API) с базой данных гардероба пользователя (RAG) и внешними API (погода). Бот анализирует контекст запроса ("питчинг, +15°C") и генерирует релевантный образ.

### 🔴 Высокая сложность (Генеративный AI, сложные ML-пайплайны и векторы)

* **Генератор принтов/кастомизации:** Интеграция Text-to-Image моделей (например, Stable Diffusion/Midjourney API) с функционалом Web2-магазина. Генерация уникального дизайна по промпту и автоматическая передача макета в печать (print-on-demand).
* **Визуальный поиск (Visual Search):** Построение пайплайна на основе эмбеддингов изображений и векторной базы данных. Нейросеть вычленяет одежду с "шумного" фото (скриншот, фото на улице) и ищет визуально похожие товары (similarity search) в каталогах.
* **Virtual Try-On (Виртуальная примерочная):** Развертывание и тюнинг тяжелых диффузионных моделей (OOTDiffusion, VTON, IDM-VTON). Требует сегментации тела, детекции ключевых точек (keypoints) и генерации изображения для реалистичного наложения одежды с учетом освещения, складок и комплекции.
